# Writings-Database — working notes

## Git workflow
- Commit directly on `master`. Do **not** create feature branches for changes in this repo.

## HTML works
- Every HTML work file must include the reader highlight script in its `<head>` (immediately before `</head>`):
  ```html
  <script src="https://historicalchristianfaith.github.io/Writings-Database/highlight.js"></script>
  ```
  When importing or adding any new HTML work, insert this line. When touching a folder, confirm existing files already have it.
